Numeric separators. You can use it to more easily write numbers with separators.

```
const products = [
  {
    name: "Sticker Pack",
    price: 10_00,
  },
  {
    name: "The Creation of React T-shirt",
    price: 30_00,
  },
  {
    name: "Expand Your Mind Crewneck",
    price: 50_00,
  },
];
```
