Q: Сколько типов данных в js вы знаете?
A: Восемь
- number    число
- bigint    гигантское число (символ n в конце числа)
- string    строка
- boolean   булевый тип
- null      специальное значение "ничего, пусто, значение неизвестно"
- undefined специальное значение "не было присвоено"
- object    объекты, для хранения структур данных. Функция тоже объект, но для них typeof вернет "function"
- symbol    уникальные идентификаторы в объектах.

Symbol('foo') === Symbol('foo')  // false
Параметр - описание символа, доступен через поле description.

Q: Сколько состояний у промиса?
